## Installation and Setup Instructions

### Prerequisites
- Windows, macOS, or Linux
- [Flutter SDK](https://flutter.dev/docs/get-started/install) (3.9 or later recommended)
- Dart 3.9 or later
- Git

### Clone the Repository
```sh
git clone https://github.com/soijukye/sandwich_shop.git
cd sandwich_shop
```

### Install Dependencies
```sh
flutter pub get
```

### Run the App
```sh
flutter run
```

## Usage Instructions

- **Select sandwich size**: Use the segmented button to choose between 6 inch and Footlong.
- **Choose bread type**: Use the dropdown menu to select your preferred bread.
